Travis CI로 VPS에 자동배포 구축하기

안녕하세요. 저는 시엘 차장과 시엘 뮤직 개발 총괄을 맡고 있는 강희원입니다. 자동 배포를 결심한 이유 시엘 뮤직은 Vue.js + Nuxt.js 기반으로 프론트 단이 구성되어 있습니다. 개발 서버를 이용해 SSR을 적용한 채로 테스트를 진행하고 싶었으나 팀 사정으로 인해 netlify라는 github page와 유사한 배포 플랫폼을 사용하고 있었습니다. 하지만 백엔드 개발이 점점 진행되어 자체 서버를 도입하기로 했고, 자동배포까지 진행하도록 결정했습니다. Travis CI 이전까지 Travis CI는 테스트 목적으로 사용했습니다. 하지만 Travis를 통한 Hugo 자동 배포를 경험해 본 이후 Travis로 vps에 배포해보고자 하는 욕심이 생겨났고 시엘뮤직에 적용하게 된 계기가 된 것 같습니다....
